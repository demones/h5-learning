# web 前端学习笔记
=======

This book help us to quickly build web front-end page. It contain html5 css3 javascript and the front-end technology for web etc.

## 开发与部署

### 安装 gitbook-cli

```bash
 npm install -g gitbook-cli
```

### 安装依赖

```bash
gitbook update
```

### 启动服务

```bash
gitbook serve
```

### 打包编译

```bash
gitbook build
```
